# SimpleNoCode
Copy-paste UI snippets for non-coders.
A free library of copy-paste UI snippets for non-coders. Add polished effects to your website - no coding required.

## What Is This?

You see a cool effect on a website but don't know how to add it? We curate simple, drop-in snippets - no build tools, no npm, just copy and paste.

## How to Use

1. Browse the snippets below
2. Click on any snippet to see the demo
3. Copy the code and paste it into your website

## Categories

- **Cursor** — Custom cursor effects
- **Text** — Animated text effects
- **Scroll** — Scroll-triggered animations
- **Buttons** — Polished button styles
- **Loaders** — Loading animations
